### API

–î–æ—Å—Ç—É–ø–Ω—ã –Ω–µ—Å–∫–æ–ª—å–∫–æ –Ω–æ–≤—ã—Ö API, —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω—ã—Ö –¥–ª—è `BigInt`.

–ì–ª–æ–±–∞–ª—å–Ω—ã–π –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä `BigInt` –∞–Ω–∞–ª–æ–≥–∏—á–µ–Ω –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä—É `Number`: –æ–Ω –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —Å–≤–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç –≤ `BigInt` (–∫–∞–∫ —É–ø–æ–º–∏–Ω–∞–ª–æ—Å—å —Ä–∞–Ω–µ–µ). –ï—Å–ª–∏ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –Ω–µ —É–¥–∞–µ—Ç—Å—è, –≤—ã–±—Ä–∞—Å—ã–≤–∞–µ—Ç—Å—è –∏—Å–∫–ª—é—á–µ–Ω–∏–µ `SyntaxError` –∏–ª–∏ `RangeError`.

–í –ø–µ—Ä–≤–æ–º –∏–∑ –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω—ã—Ö –ø—Ä–∏–º–µ—Ä–æ–≤ –ø–µ—Ä–µ–¥–∞–µ—Ç—Å—è —á–∏—Å–ª–æ–≤–æ–π –ª–∏—Ç–µ—Ä–∞–ª –≤ `BigInt()`. –≠—Ç–æ –ø–ª–æ—Ö–∞—è –ø—Ä–∞–∫—Ç–∏–∫–∞, —Ç–∞–∫ –∫–∞–∫ `Number` —Å—Ç—Ä–∞–¥–∞—é—Ç –æ—Ç –ø–æ—Ç–µ—Ä–∏ —Ç–æ—á–Ω–æ—Å—Ç–∏, –∏ –º—ã —É–∂–µ –º–æ–∂–µ–º –ø–æ—Ç–µ—Ä—è—Ç—å —Ç–æ—á–Ω–æ—Å—Ç—å –¥–æ —Ç–æ–≥–æ, –∫–∞–∫ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤ `BigInt`:
