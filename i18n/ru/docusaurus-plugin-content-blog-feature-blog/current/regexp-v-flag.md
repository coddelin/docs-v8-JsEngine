```js
re.test('üë®üèæ‚Äç‚öïÔ∏è'); // '\u{1F468}\u{1F3FE}\u200D\u2695\uFE0F'
// ‚Üí true ‚úÖ
```

–≠—Ç–æ—Ç —Ñ—Ä–∞–≥–º–µ–Ω—Ç –∫–æ–¥–∞ –æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ —Å–≤–æ–π—Å—Ç–≤—É —Å—Ç—Ä–æ–∫ `RGI_Emoji`, –∫–æ—Ç–æ—Ä–æ–µ Unicode –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –∫–∞–∫ ¬´–ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ –≤—Å–µ—Ö –¥–æ–ø—É—Å—Ç–∏–º—ã—Ö —ç–º–æ–¥–∑–∏ (—Å–∏–º–≤–æ–ª–æ–≤ –∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π), —Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞–Ω–Ω—ã—Ö –¥–ª—è –æ–±—â–µ–≥–æ –æ–±–º–µ–Ω–∞¬ª. –° —ç—Ç–∏–º —Ç–µ–ø–µ—Ä—å –º—ã –º–æ–∂–µ–º —Å–æ–ø–æ—Å—Ç–∞–≤–ª—è—Ç—å —ç–º–æ–¥–∑–∏ –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç —Ç–æ–≥–æ, —Å–∫–æ–ª—å–∫–æ –∫–æ–¥–æ–≤—ã—Ö —Ç–æ—á–µ–∫ –æ–Ω–∏ —Å–æ–¥–µ—Ä–∂–∞—Ç!

–§–ª–∞–≥ `v` –≤–∫–ª—é—á–∞–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∫—É —Å–ª–µ–¥—É—é—â–∏—Ö Unicode —Å–≤–æ–π—Å—Ç–≤ —Å—Ç—Ä–æ–∫ –∏–∑–Ω–∞—á–∞–ª—å–Ω–æ:

- `Basic_Emoji`
- `Emoji_Keycap_Sequence`
- `RGI_Emoji_Modifier_Sequence`
- `RGI_Emoji_Flag_Sequence`
- `RGI_Emoji_Tag_Sequence`
- `RGI_Emoji_ZWJ_Sequence`
- `RGI_Emoji`

–≠—Ç–æ—Ç —Å–ø–∏—Å–æ–∫ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ–º—ã—Ö —Å–≤–æ–π—Å—Ç–≤ –º–æ–∂–µ—Ç —Ä–∞—Å—à–∏—Ä–∏—Ç—å—Å—è –≤ –±—É–¥—É—â–µ–º, –µ—Å–ª–∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç Unicode –æ–ø—Ä–µ–¥–µ–ª–∏—Ç –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ —Å—Ç—Ä–æ–∫. –•–æ—Ç—è –≤—Å–µ —Ç–µ–∫—É—â–∏–µ —Å–≤–æ–π—Å—Ç–≤–∞ —Å—Ç—Ä–æ–∫ —Å–≤—è–∑–∞–Ω—ã —Å —ç–º–æ–¥–∑–∏, –±—É–¥—É—â–∏–µ —Å–≤–æ–π—Å—Ç–≤–∞ –º–æ–≥—É—Ç —Å–ª—É–∂–∏—Ç—å —Å–æ–≤–µ—Ä—à–µ–Ω–Ω–æ –¥—Ä—É–≥–∏–º —Ü–µ–ª—è–º.

:::note
**–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ:** –•–æ—Ç—è —Å–≤–æ–π—Å—Ç–≤–∞ —Å—Ç—Ä–æ–∫ –≤ –Ω–∞—Å—Ç–æ—è—â–µ–µ –≤—Ä–µ–º—è –æ–≥—Ä–∞–Ω–∏—á–µ–Ω—ã –Ω–æ–≤—ã–º —Ñ–ª–∞–≥–æ–º `v`, [–º—ã –ø–ª–∞–Ω–∏—Ä—É–µ–º –≤ –∫–æ–Ω–µ—á–Ω–æ–º –∏—Ç–æ–≥–µ —Å–¥–µ–ª–∞—Ç—å –∏—Ö –¥–æ—Å—Ç—É–ø–Ω—ã–º–∏ –∏ –≤ —Ä–µ–∂–∏–º–µ `u`](https://github.com/tc39/proposal-regexp-v-flag/issues/49).

## –ù–æ—Ç–∞—Ü–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ + —Å–∏–Ω—Ç–∞–∫—Å–∏—Å —Å—Ç—Ä–æ–∫–æ–≤—ã—Ö –ª–∏—Ç–µ—Ä–∞–ª–æ–≤

–ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å —ç–∫—Ä–∞–ø–∞–º–∏ `\p{‚Ä¶}` (–±—É–¥—å —Ç–æ —Å–≤–æ–π—Å—Ç–≤–∞ —Å–∏–º–≤–æ–ª–æ–≤ –∏–ª–∏ –Ω–æ–≤—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ —Å—Ç—Ä–æ–∫) –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–µ–∑–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ä–∞–∑–Ω–æ—Å—Ç—å/–≤—ã—á–∏—Ç–∞–Ω–∏–µ –∏–ª–∏ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ. –° —Ñ–ª–∞–≥–æ–º `v` —Ç–µ–ø–µ—Ä—å –º–æ–∂–Ω–æ –≤–∫–ª–∞–¥—ã–≤–∞—Ç—å –∫–ª–∞—Å—Å—ã —Å–∏–º–≤–æ–ª–æ–≤, –∏ —ç—Ç–∏ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ –º–æ–∂–Ω–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å –≤–Ω—É—Ç—Ä–∏ –Ω–∏—Ö, –∞ –Ω–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Å–º–µ–∂–Ω—ã—Ö —É—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–π –ø—Ä–µ–¥–≤–æ—Å—Ö–∏—â–µ–Ω–∏—è –∏–ª–∏ —Ä–µ—Ç—Ä–æ—Å–ø–µ–∫—Ç–∏–≤—ã –∏–ª–∏ –¥–ª–∏–Ω–Ω—ã—Ö –∫–ª–∞—Å—Å–æ–≤ —Å–∏–º–≤–æ–ª–æ–≤, –≤—ã—Ä–∞–∂–∞—é—â–∏—Ö –≤—ã—á–∏—Å–ª–µ–Ω–Ω—ã–µ –¥–∏–∞–ø–∞–∑–æ–Ω—ã.

### –†–∞–∑–Ω–æ—Å—Ç—å/–≤—ã—á–∏—Ç–∞–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é `--`

–°–∏–Ω—Ç–∞–∫—Å–∏—Å `A--B` –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å—Ç—Ä–æ–∫ _–≤ `A`, –Ω–æ –Ω–µ –≤ `B`_, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è —Ä–∞–∑–Ω–æ—Å—Ç—å/–≤—ã—á–∏—Ç–∞–Ω–∏–µ.

–ù–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ –µ—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ —Å–æ–ø–æ—Å—Ç–∞–≤–∏—Ç—å –≤—Å–µ –≥—Ä–µ—á–µ—Å–∫–∏–µ —Å–∏–º–≤–æ–ª—ã, –∫—Ä–æ–º–µ –±—É–∫–≤—ã `œÄ`? –ò—Å–ø–æ–ª—å–∑—É—è –Ω–æ—Ç–∞—Ü–∏—é –º–Ω–æ–∂–µ—Å—Ç–≤–∞, —Ä–µ—à–∏—Ç—å —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ:

```js
/[\p{Script_Extensions=Greek}--œÄ]/v.test('œÄ'); // ‚Üí false
```

–ò—Å–ø–æ–ª—å–∑—É—è `--` –¥–ª—è —Ä–∞–∑–Ω–æ—Å—Ç–∏/–≤—ã—á–∏—Ç–∞–Ω–∏—è, –¥–≤–∏–∂–æ–∫ —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Å–ª–æ–∂–Ω—É—é —Ä–∞–±–æ—Ç—É –∑–∞ –≤–∞—Å, —Å–æ—Ö—Ä–∞–Ω—è—è –≤–∞—à –∫–æ–¥ —á–∏—Ç–∞–µ–º—ã–º –∏ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ–º—ã–º.

–ß—Ç–æ –µ—Å–ª–∏, –≤–º–µ—Å—Ç–æ –æ–¥–Ω–æ–≥–æ —Å–∏–º–≤–æ–ª–∞, –º—ã —Ö–æ—Ç–∏–º –≤—ã—á–µ—Å—Ç—å –Ω–∞–±–æ—Ä —Å–∏–º–≤–æ–ª–æ–≤ `Œ±`, `Œ≤` –∏ `Œ≥`? –ù–µ—Ç –ø—Ä–æ–±–ª–µ–º ‚Äî –º—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤–ª–æ–∂–µ–Ω–Ω—ã–π –∫–ª–∞—Å—Å —Å–∏–º–≤–æ–ª–æ–≤ –∏ –≤—ã—á–µ—Å—Ç—å –µ–≥–æ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ:

```js
/[\p{Script_Extensions=Greek}--[Œ±Œ≤Œ≥]]/v.test('Œ±'); // ‚Üí false
/[\p{Script_Extensions=Greek}--[Œ±-Œ≥]]/v.test('Œ≤'); // ‚Üí false
```

–î—Ä—É–≥–æ–π –ø—Ä–∏–º–µ—Ä ‚Äî —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –Ω–µ-ASCII —Ü–∏—Ñ—Ä, –Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è –∏—Ö –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –≤ ASCII —Ü–∏—Ñ—Ä—ã –ø–æ–∑–∂–µ:

```js
/[\p{Decimal_Number}--[0-9]]/v.test('ëúπ'); // ‚Üí true
/[\p{Decimal_Number}--[0-9]]/v.test('4'); // ‚Üí false
```

–ù–æ—Ç–∞—Ü–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∞ —Å –Ω–æ–≤—ã–º–∏ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏ —Å—Ç—Ä–æ–∫:

```js
// –ü—Ä–∏–º–µ—á–∞–Ω–∏–µ: üè¥ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ 7 –∫–æ–¥–æ–≤—ã—Ö —Ç–æ—á–µ–∫.

/^\p{RGI_Emoji_Tag_Sequence}$/v.test('üè¥'); // ‚Üí true
/^[\p{RGI_Emoji_Tag_Sequence}--\q{üè¥}]$/v.test('üè¥'); // ‚Üí false
```

–≠—Ç–æ—Ç –ø—Ä–∏–º–µ—Ä —Å–æ–ø–æ—Å—Ç–∞–≤–ª—è–µ—Ç –ª—é–±—É—é –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ç–µ–≥–æ–≤ —ç–º–æ–¥–∑–∏ RGI, _–∫—Ä–æ–º–µ_ —Ñ–ª–∞–≥–∞ –®–æ—Ç–ª–∞–Ω–¥–∏–∏. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ `\q{‚Ä¶}`, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è –Ω–æ–≤—ã–º —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–æ–º –¥–ª—è —Å—Ç—Ä–æ–∫–æ–≤—ã—Ö –ª–∏—Ç–µ—Ä–∞–ª–æ–≤ –≤–Ω—É—Ç—Ä–∏ –∫–ª–∞—Å—Å–æ–≤ —Å–∏–º–≤–æ–ª–æ–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, `\q{a|bc|def}` —Å–æ–ø–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Å—Ç—Ä–æ–∫–∏ `a`, `bc`, –∏ `def`. –ë–µ–∑ `\q{‚Ä¶}` –Ω–µ–ª—å–∑—è –±—ã–ª–æ –±—ã –≤—ã—á–µ—Å—Ç—å –∂–µ—Å—Ç–∫–æ –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –º–Ω–æ–≥–æ—Å–∏–º–≤–æ–ª—å–Ω—ã–µ —Å—Ç—Ä–æ–∫–∏.

### –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ —Å `&&`

–°–∏–Ω—Ç–∞–∫—Å–∏—Å `A&&B` —Å–æ–ø–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Å—Ç—Ä–æ–∫–∏, –∫–æ—Ç–æ—Ä—ã–µ _–µ—Å—Ç—å –∏ –≤ `A`, –∏ –≤ `B`_, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –¥–µ–ª–∞—Ç—å, –Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≥—Ä–µ—á–µ—Å–∫–∏—Ö –±—É–∫–≤:

```js
const re = /[\p{Script_Extensions=Greek}&&\p{Letter}]/v;
// U+03C0 –ú–ê–õ–ê–Ø –ì–†–ï–ß–ï–°–ö–ê–Ø –ë–£–ö–í–ê –ü–ò
re.test('œÄ'); // ‚Üí true
// U+1018A –ì–†–ï–ß–ï–°–ö–ò–ô –ù–£–õ–ï–í–û–ô –ó–ù–ê–ö
re.test('êÜä'); // ‚Üí false
```

–°–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤—Å–µ—Ö –ø—Ä–æ–±–µ–ª–æ–≤ ASCII:

```js
const re = /[\p{White_Space}&&\p{ASCII}]/v;
re.test('\n'); // ‚Üí true
re.test('\u2028'); // ‚Üí false
```

–ò–ª–∏ —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤—Å–µ—Ö –º–æ–Ω–≥–æ–ª—å—Å–∫–∏—Ö —á–∏—Å–µ–ª:

```js
const re = /[\p{Script_Extensions=Mongolian}&&\p{Number}]/v;
// U+1817 –ú–û–ù–û–õ–¨–°–ö–ê–Ø –¶–ò–§–†–ê –°–ï–ú–¨
re.test('·†ó'); // ‚Üí true
// U+1834 –ë–£–ö–í–ê –ú–û–ù–û–õ–¨–°–ö–û–ô –ß–ê
re.test('·†¥'); // ‚Üí false
```

### –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ

–°–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫, –∫–æ—Ç–æ—Ä—ã–µ _–ª–µ–∂–∞—Ç –≤ A –∏–ª–∏ –≤ B_, —É–∂–µ –±—ã–ª–æ –≤–æ–∑–º–æ–∂–Ω–æ –¥–ª—è –æ–¥–∏–Ω–æ—á–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ —Å—Ç—Ä–æ–∫, –∏—Å–ø–æ–ª—å–∑—É—è –∫–ª–∞—Å—Å —Å–∏–º–≤–æ–ª–æ–≤, –Ω–∞–ø—Ä–∏–º–µ—Ä `[\p{Letter}\p{Number}]`. –° —Ñ–ª–∞–≥–æ–º `v` —ç—Ç–∞ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –±–æ–ª–µ–µ –º–æ—â–Ω–æ–π, –ø–æ—Å–∫–æ–ª—å–∫—É —Ç–µ–ø–µ—Ä—å –æ–Ω–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –∫–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–∞ —Å —Å–≤–æ–π—Å—Ç–≤–∞–º–∏ —Å—Ç—Ä–æ–∫ –∏–ª–∏ —Å—Ç—Ä–æ–∫–æ–≤—ã–º–∏ –ª–∏—Ç–µ—Ä–∞–ª–∞–º–∏:

```js
const re = /^[\p{Emoji_Keycap_Sequence}\p{ASCII}\q{üáßüá™|abc}xyz0-9]$/v;

re.test('4Ô∏è‚É£'); // ‚Üí true
re.test('_'); // ‚Üí true
re.test('üáßüá™'); // ‚Üí true
re.test('abc'); // ‚Üí true

re.test('4'); // ‚Üí true
```

–ö–ª–∞—Å—Å —Å–∏–º–≤–æ–ª–æ–≤ –≤ —ç—Ç–æ–º —à–∞–±–ª–æ–Ω–µ –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç:

- —Å–≤–æ–π—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫–∏ (`\p{Emoji_Keycap_Sequence}`)
- —Å–≤–æ–π—Å—Ç–≤–æ —Å–∏–º–≤–æ–ª–∞ (`\p{ASCII}`)
- —Å–∏–Ω—Ç–∞–∫—Å–∏—Å —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ –ª–∏—Ç–µ—Ä–∞–ª–∞ –¥–ª—è –º–Ω–æ–≥–æ–∫–æ–¥–æ–≤—ã—Ö —Ç–æ—á–µ–∫ —Å—Ç—Ä–æ–∫ `üáßüá™` –∏ `abc`
- –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –∫–ª–∞—Å—Å–∞ —Å–∏–º–≤–æ–ª–æ–≤ –¥–ª—è –æ–¥–∏–Ω–æ—á–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ `x`, `y` –∏ `z`
- –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –∫–ª–∞—Å—Å–∞ —Å–∏–º–≤–æ–ª–æ–≤ –¥–ª—è –¥–∏–∞–ø–∞–∑–æ–Ω–∞ —Å–∏–º–≤–æ–ª–æ–≤ –æ—Ç `0` –¥–æ `9`

–î—Ä—É–≥–æ–π –ø—Ä–∏–º–µ—Ä ‚Äî —ç—Ç–æ —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤—Å–µ—Ö —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã—Ö —Ñ–ª–∞–∂–∫–æ–≤-—ç–º–æ–¥–∑–∏, –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç —Ç–æ–≥–æ, –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω—ã –ª–∏ –æ–Ω–∏ –∫–∞–∫ –¥–≤—É—Ö–±—É–∫–≤–µ–Ω–Ω—ã–π –∫–æ–¥ ISO (`RGI_Emoji_Flag_Sequence`) –∏–ª–∏ –∫–∞–∫ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ç–µ–≥–æ–≤ —Å–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º —Å–ª—É—á–∞–µ–º (`RGI_Emoji_Tag_Sequence`):

```js
const reFlag = /[\p{RGI_Emoji_Flag_Sequence}\p{RGI_Emoji_Tag_Sequence}]/v;
// –ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ñ–ª–∞–≥–∞, —Å–æ—Å—Ç–æ—è—â–∞—è –∏–∑ 2 –∫–æ–¥–æ–≤—ã—Ö —Ç–æ—á–µ–∫ (—Ñ–ª–∞–≥ –ë–µ–ª—å–≥–∏–∏):
```
