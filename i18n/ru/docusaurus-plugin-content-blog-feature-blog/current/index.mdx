<h1>Функции JavaScript и WebAssembly</h1>

<p>Этот раздел объясняет последние новые функции в <strong>JavaScript</strong>, языке программирования, стандартизованном <a href="https://tc39.es/ecma262/">ECMA-262 (ECMAScript)</a>
и <a href="https://tc39.es/ecma402/">ECMA-402 (API интернационализации ECMAScript)</a>, а также в <a href="https://webassembly.github.io/spec/core/"><strong>WebAssembly</strong></a> (Wasm).</p>
