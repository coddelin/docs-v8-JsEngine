<h1>Recursos de JavaScript e WebAssembly</h1>

<p>Esta seção explica os mais recentes novos recursos em <strong>JavaScript</strong>, a linguagem de programação especificada pelo <a href="https://tc39.es/ecma262/">ECMA-262 (ECMAScript)</a>
e pela <a href="https://tc39.es/ecma402/">ECMA-402 (API de Internacionalização do ECMAScript)</a>, e em <a href="https://webassembly.github.io/spec/core/"><strong>WebAssembly</strong></a> (Wasm).</p>

<!--truncate-->
<p>Nosso objetivo é fornecer explicações <strong>concisas</strong> sobre os novos recursos da linguagem com <strong>exemplos de código fáceis de entender</strong>.
Para documentação mais aprofundada, consulte os links externos em cada explicação.</p>
