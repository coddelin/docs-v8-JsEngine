## ARM

간단한 ARM 포트의 경우 직접 처리하세요. 문제가 있거나 질문이 있을 경우, 검토자 또는 CC 목록에 `v8-arm-ports@googlegroups.com`를 추가하여 CL에서 ARM 팀에 연락하세요. 질문이 있을 경우 해당 그룹에 이메일을 보낼 수 있습니다 ([mailto link](mailto:v8-arm-ports@googlegroups.com)).
