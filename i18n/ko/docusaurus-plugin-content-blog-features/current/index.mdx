<h1>JavaScript와 WebAssembly 기능</h1>

<p>이 섹션은 <strong>JavaScript</strong>(<a href="https://tc39.es/ecma262/">ECMA-262 (ECMAScript)</a>
및 <a href="https://tc39.es/ecma402/">ECMA-402 (ECMAScript 국제화 API)</a>로 명시된 프로그래밍 언어)의 최신 기능과, <a href="https://webassembly.github.io/spec/core/"><strong>WebAssembly</strong></a> (Wasm)의 새로운 기능에 대해 설명합니다.</p>

<!--truncate-->
<p>우리는 새로운 언어 기능에 대해 <strong>간결한</strong> 설명과 <strong>이해하기 쉬운 코드 예제</strong>를 제공하고자 노력합니다.
더 깊이 있는 문서를 확인하려면 각 설명서의 외부 링크를 참조하십시오.</p>
